Minicell Apparel

"Web-based E-Commerce application"

Introduction

Minicell is a simple yet functional e-commerce web application. The system allows users to browse and purchase clothing items, while administrators can manage product listings. Designed with the MVC (Model-View-Controller) architecture and developed using PHP, Minicell demonstrates practical application of web development principles in a real-world context.

Features

  • User registration and login authentication
  • Product browsing, adding products to cart
  • Responsive and dynamic product display
  • Admin dashboard for product management

How it works?

  1. User registers or logs in
  2. User browses the available products
  3. Clicking a product displays detailed information
  4. User can add products to cart or proceed to checkout
  5. Admins can log in and manage the product list (add, update, delete)
